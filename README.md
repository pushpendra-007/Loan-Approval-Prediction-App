# Loan Approval Prediction App

This is a Machine Learning based web application built using Python and Streamlit.
It predicts whether a loan will be approved or not based on user inputs.

## Features
- Applicant Income
- Co-applicant Income
- Loan Amount
- Loan Term
- Education & Marital Status

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## Output
The model predicts loan approval status.
